## 章节导言

目标很明确：用最少的节点实现一条稳定、可扩展的文本到图片工作流。这条流水线要做到：随时修改提示词、调整画幅、保存成品，不再依赖别人共享的成品图摸索。

![文生图工作流示例](/ComfyUI-Beginners-Guide/assets/uploads/wechat_2025-10-04_193054_297.png "Qwen-Image 文生图工作流")

<TipCard title="动手前必备检查" tone="info">
  <ul>
    <li>确认模型文件夹无中文路径，文件名与 ComfyUI 设置一致。</li>
    <li>提前写好正向/反向提示词，避免边连线边纠结文案。</li>
    <li>把输出文件夹清空或备份，方便定位最新成品。</li>
  </ul>
</TipCard>

## 阶段一：筑好基础积木

<StepCard index={1} title="加载模型底座" summary="模型加载：UNET、LoRA、CLIP、VAE" id="step-1">
  <ol>
    <li>双击搜索 <code>unet加载器</code>等节点， 并双击添加。</li>
    <li>分别选择unet、lora、clip、vae配套的模型（例如 <code>qwen_image_fp8_e4m3fn</code>）,</li>
  </ol>
</StepCard>

![](/ComfyUI-Beginners-Guide/assets/uploads/wechat_2025-10-05_122119_024.png)

<StepCard index={2} title="准备正负提示词" summary="CLIP文本编码器、条件零化" id="step-2">
  <ol>
    <li>添加<code>CLIP文本编码器</code> 节点作为正向提示词。</li>
    <li>右侧<code>条件</code>接入<code>条件零化</code>节点作为“反向提示词”。</li>
    <li>将 <code> CLIP加载器</code> 输出接入节点的 <code>clip</code> 端口。</li>
    <li>

      </div>
    </li>
  </ol>
</StepCard>

![](/ComfyUI-Beginners-Guide/assets/uploads/wechat_2025-10-05_124132_373.png)

<StepCard index={3} title="建立潜空间画布" summary="空Latent" id="step-3">
  <ol>
    <li>添加 <code>空Latent</code> 节点，设置宽高为 1024 × 1536，作为输出图片的尺寸。</li>
    <li>保持批次大小为 1，代表单次生成一张图。</li>
    <li>后续改画幅时只需修改这里的宽高，其他节点无需调整。</li>
  </ol>
</StepCard>

## 阶段二：闭合生成核心

至此，`K采样器`前的节点已经设置完毕，接下就是**采样去噪**环节。

<StepCard index={4} title="配置 K采样器" summary="四条输入缺一不可" id="step-4">
  <ol>
    <li>连线：<code>Unet+LoRA / MODEL串联 → K采样器 / model</code>。（<code>模型采样算法AuraFlow</code>这里不做展开，直接复制就好）</li>
    <li>连线：正向、反向提示词分别进入 <code>正向条件</code> 与 <code>负向条件</code>。</li>
    <li>连线：<code>空Latent / LATENT → K采样器 / latent</code>。</li>
    <li>参数建议：<code>Steps 4</code>、<code>CFG 1</code>、采样器选择 <code>euler</code>
、调度器选择<code>simple</code>。（注，4步是因为加速LoRA的作用，否则可能需要20-30步）</li>
  </ol>
</StepCard>

![](/ComfyUI-Beginners-Guide/assets/uploads/wechat_2025-10-05_130249_357.png)

<StepCard index={5} title="解码并保存" summary="VAE解码 + 保存图片" id="step-5">
  <ol>
    <li>添加 <code>VAE解码</code> 与 <code>保存图片</code> 节点。</li>
    <li>将 <code>K采样器 / LATENT</code> 接到 <code>VAE解码 / latent</code>，再将 <code>VAE加载器</code> 接到 <code>vae</code> 端口。</li>
    <li>最后把 <code>VAE解码 / 图像</code> 输出连到 <code>保存图像</code> 的输入。</li>
  </ol>
</StepCard>

![](/ComfyUI-Beginners-Guide/assets/uploads/wechat_2025-10-05_130936_692.png)

<TipCard title="快速验证任务流" tone="success">
  <ol>
    <li>点击右上角 <strong>运行</strong>，确认控制台无红色报错。</li>
    <li>在 <strong>任务列表</strong> 找到最新图片，检查是否符合预期提示词。</li>
    <li>想重现图像，记录采样器下方的 Seed 值，下次改为固定模式即可复现。</li>
  </ol>
</TipCard>

![](/ComfyUI-Beginners-Guide/assets/uploads/wechat_2025-10-05_131524_185.png)

## 阶段三：一套模板，多种玩法

* **调整画幅**：修改 空latent 的宽高，建议保持 64 的倍数避免报错。
* **锁定风格**：将 K采样器 的随机模式改为 <code>固定</code>，输入固定数值即可复刻。
