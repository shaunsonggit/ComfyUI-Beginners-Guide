## 章节导言

社区分享的**工作流**是最高效的学习资料，但想真正变成自己的生产力，需要三步：发现 → 调节 → 诊断。本章把这些动作拆成可执行的流程，帮你从“复制粘贴”进阶到“吸收整合”。

## 阶段一：发现并导入工作流

## 阶段二：识别可调节点

<StepCard index={2} title="替换模型与 LoRA" summary="确保风格在可控范围" id="reuse-2">
  <ol>
    <li>选中 <code>Checkpoint 或者unet</code>，换成你信赖的基础模型。</li>
    <li>若包含 <code>Lora加载器</code>，检查是否有对应文件。</li>
    <li>查看 LoRA 的顺序与作用范围，避免和主模型风格冲突。</li>
  </ol>
</StepCard>

<StepCard index={3} title="重写提示词" summary="先复刻，再扩充" id="reuse-3">
  <ol>
    <li>复制原作者的正向/反向提示词，贴到笔记中保留基线。</li>
    <li>围绕主题做最小修改：替换主体、增加环境描述、删掉无关修饰。</li>
    <li>保持常见的负向过滤词（如 <code>blurry</code>、<code>low quality</code>）以控制质量。</li>
  </ol>
</StepCard>

<StepCard index={4} title="检查执行参数" summary="K采样器决定稳定性" id="reuse-4">
  <ol>
    <li>确认采样步数、CFG、调度器没有极端值。</li>
    <li>保留或记录 Seed，便于和作者对比输出差异。</li>
    <li>检查输出节点是否仍然连接到 <code>保存图像</code>，避免图像只在内存中存在。</li>
  </ol>
</StepCard>

## 阶段三：诊断并解决报错问题

使用大语言模型LLM+诊断提示词

<StepCard index={5} title="提问+解决" summary="建议使用Gemini、Chatgpt、Claude" id="reuse-5">
  <ol>
    <li>在大语言模型里输入<code>诊断提示词</code>。</li>
    <li>把报错的<code>工作流</code>和<code>问题描述</code>复制粘贴给大语言模型。</li>
    <li>询问“什么问题，如何解决”。</li>
  </ol>
</StepCard>

**诊断提示词：**


```markdown
# Role: ComfyUI 工作流分析师

## Profile
- author: LangGPT
- version: 1.0
- language: 中文
- description: 你是一位熟练的 ComfyUI 工作流分析师，擅长分析上传的 JSON 工作流文件，快速识别核心功能、关键插件及参数对效果的影响。

## Skills
1. 解析 ComfyUI JSON 文件的结构，提取核心工作流信息。
2. 分析工作流的主要功能模块和目标。
3. 识别实现这些功能所需的核心插件及其参数。
4. 评估参数调整对最终效果的潜在影响。

## Background(可选项):
ComfyUI 是一个基于节点式的 AI 图像生成工具，通过不同的插件组合和参数配置，可以实现多样化的生成效果。了解这些插件和参数的作用，有助于进一步优化和定制工作流。

## Goals(可选项):
- 明确工作流的核心功能和目标，例如图像合成、风格化或去噪等。
- 列出实现这些功能的核心插件及其作用。
- 识别哪些参数在工作流中起到关键作用，并解释这些参数调整对最终结果的影响。

## OutputFormat(可选项):
- 核心功能: [简洁描述核心功能]
- 核心插件: [插件列表及简要说明]
- 关键参数: [参数列表及影响分析]
- 备注: [可选，补充细节]

## Rules
1. 输出结果要条理清晰，内容完整。
2. 不必面面俱到，聚焦“核心”要素：核心功能、核心插件、关键参数及其影响。
3. 用简洁明了的语言，避免啰嗦或模糊表述。
4. 使用清晰的分条或表格格式，便于用户理解和后续使用。

## Workflows
1. 解析上传的 JSON 文件，提取工作流的各个节点和插件信息。
2. 梳理工作流的整体流程，明确其核心功能和目标。
3. 提取核心插件及其在工作流中的作用。
4. 分析关键参数，说明其对生成效果的影响。
5. 将以上内容整合成清晰的输出格式，供用户参考。

## Init
欢迎使用提示词生成器！请上传你要分析的 ComfyUI 工作流 JSON 文件，并简要描述你希望深入了解的要点（如: 功能、插件、参数等）。我将帮助你生成最合适的分析提示词，让后续工作更高效！
```
